# About this repository

This repository contains my submissions for assignments in the Brandeis GPS Bioinformatics Master's program. I chose a subset of all assignments that I feel represent the quality of my work, as well as my dedication to scientific curiosity and learning. Some of these files are PDFs, but many are self-contained HTML reports generated via RMarkdown or a comparable report generation scheme using Python. Most reports contain a combination of code, results, and interpretation with citations.

Due to the format of the reports, GitHub's language scanning algorithm mostly detects HTML. That said, the code demonstrated in them is written in the following: 
+ **R** 
+ **Bash**
+ **MATLAB**
+ **Python**

This repository does **not** contain anything from my capstone research project in the bioinformatics program (RBIF 120: Research Topics in Computational Biology). For that, please instead see [Helix-EXon-finder](https://github.com/GreenMtn-bioinfo/Helix-EXon-finder) and [HEX-finder-training](https://github.com/GreenMtn-bioinfo/RBIF120-HEX-finder-training). My [sequence analysis final project](/Final%20Projects/RBIF109%20DNA%20Structural%20Signatures%20at%20Splice%20Sites%20-%20Jonathan%20Hummel.html) involved some background work that led to my capstone, but it represents a much simpler implementation of the structural profile generation algorithm used in the two repositories mentioned above.

To view any of the files, you will need to download and open them in a PDF reader or web browser. Some HTML files are large and can take a few moments to load completely in the web browser.
