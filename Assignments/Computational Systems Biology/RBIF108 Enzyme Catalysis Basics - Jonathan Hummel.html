1) Enzyme-catalyzed reaction

With the help of this week's tutorials and reading materials, I put the system of mass action ODEs representing a basic enzyme-catalyzed conversion of a substrate to a product into a MATLAB function [1-5]. This code can be seen directly below. It is worth noting that all four rates of change were included in in the ODE function, even though not all are needed to solve the system. This was done purely out of convinience, as it is easier to simply get all solutions from one function and it did not noticeably affect the numerical convergence time here.

EnzymeSubstrateProduct.m

function dxdt = EnzymeSubstrateProduct(t, x, params)
% EnzymeSubstrateProduct: This functions contains a system of
% ODEs that describe a simple enzyme (E) -catalyzed reaction
% that converts a substrate (S) into a product (P):
% E + S <-> ES -> E + P
% Formulated using the law of mass action. Rate constants kf and kr
% describe the first (reversible) reaction and kcat describes the
% second reaction that makes product. E0 is initial free enzyme concentration.
% 'params' contains rate constants and E0 (kf, kr, kcat, E0), while
% 'x' provides the state variables as inputs (i.e., E, S, P, ES)
% conservation of mass is used to constrain the enzyme-related concentrations
 
% For interpretabillity, assign the rate constants from the 'params' input
kf = params(1); % ES complex forward
kr = params(2); % ES complex reverse
kcat = params(3); % Substrate converted to product (enzyme-catalyzed)
Eo = params(4); % Initial concentration of free enzyme
 
% For interpretabillity, reassign state variable names
E = x(1);
S = x(2);
% P = x(3); % not used in system definition below
% ES = x(4); % not needed, ES constrained by conservation of mass below
% Define the full system of differential equations
% Note that not all of these equations are technically needed to solve
% the system numerically, but all were included here for convinience
ES = Eo - E; % total enzyme present in any form remains constant
dEdt = kr*ES - kf*E*S + kcat*ES;
dSdt = kr*ES - kf*E*S;
dPdt = kcat*ES;
dESdt = kf*E*S - kr*ES - kcat*ES;
dxdt = [dEdt; dSdt; dPdt; dESdt];
end
Below is the script I wrote to solve the system defined above and plot the results. It is largely based on the code from the Week 5 Assignment:

RunEnzymeSubstrateProduct.m

%% Start with a blank slate
close all
 
%% Set the initial conditions for the ODE system, as well as the model parameters
p(1)=0.5; kf = p(1);
p(2)=0.5; kr = p(2);
p(3)=0.5; kcat = p(3);
p(4)=1; E0 = p(4); % initial concentration of enzyme at t=0, constrains species containing E
S0 = 100;
IC = [E0 S0 0 0]; % E, S, P, and ES at t=0
 
%% Set the convergence tolerance constraints for the ODE solver
options=odeset;
options.AbsTol=1e-6;
options.RelTol=1e-6;
 
%% Anonymize the function call, and assign the parameter set to this anonymized function
f=@(t,x)EnzymeSubstrateProduct(t,x,p);
 
%% Call the numerical ODE solver
Trange = [0 1000];
[T, X] = ode15s(f, Trange, IC, options);
 
%% Lets add some reference points from Schnell & Mendoza, 1997
KM = (kr + kcat)/kf; % (kr+kcat)/kf
tc = 1/(kf*(KM+S0)); % Transient ends/saturation time (Schnell & Mendoza, 1997)
ts = (KM+S0)/(kcat*E0); % "Slow" time-scale over which notable change in S occurs (Schnell & Mendoza, 1997)
% This should be << 1 for QSS assumption to hold (Schnell & Mendoza, 1997)
QSSA_criteria = E0/(S0+KM)

The 3 figures above are all of the same numerical solution, but show different time windows/resolutions within it. All plots have tc and/or ts, as defined in this week's paper by Schnell & Mendoza [3], indicated using vertical lines for reference. Initially, for roughly t < tc, the system is in its fast transient stage, during which the free enzyme is rapidly bound by substrate until it is saturated (see middle plot) [3]. As seen above, for the chosen parameters and values of E0 and S0, this initial transient occurs very quickly and tc is not even visible in the first plot of the entire reaction. The second time, ts, defines the slow timescale over which a notable change in the substrate concentration occurs gradually [3]. It is within the window of tc < t < ts that the quasi-steady state assumption holds most strongly, and so this is where the Michaelis-Menten approximation is most accurate (farther away from these boundaries is better). As noted by Schnell & Mendoza, the quasi-steady state assumption is valid when E0/(KM+S0) << 1 [3]. This value was calculated and printed above (QSSA_criteria), and it can be seen that this condition is more or less met by the parameters and initial conditions chosen. It is because of the relatively large value of S0 that the initial transient so short and insignificant, i.e. the enzyme saturates rapidly and this barely changes [S] from S0. Also, ts is reasonably long relative to the time it takes for the reaction to complete due to the ratio of S0 to E0, assuming all else remains unchanged (i.e. rate constants). In the third plot, it can be seen that for t > ts (and even leading up to this), assumptions upon which the Michaelis-Menten approximation is based begin to erode somewhat. For example, the substrate concentration is no longer much greater than the enzyme concentration, and the complex concentration begins to change more quickly such that d[ES]/dt ≈ 0 does not hold as strongly as it did for tc < t < ts.

2) Michaelis-Menten accuracy

While there is strong indication above that the Michaelis-Menten approximation works reasonably well for the rate constants and initial conditions chosen [3], this can be shown more explicitly by directly comparing the output of the Michaelis-Menten equation to the exact numerical solution. This was done using the function and script shown below. While there are several ways to acheive the same goal, out of convinience, I have chosen to simply solve a version of the Michaelis-Menten equation numerically for [S] using the same rate constants and initial conditions as above.

MichaelisMenten.m

function dSdt = MichaelisMenten(t, s, params)
% MichaelisMenten: approximation of basic enzyme catalyzed reaction
% system expressed in the EnzymeSubstrateProduct() function:
% E + S <-> ES -> E + P
% Conservation of mass and quasi-steady state assumptions are used to
% reduce the system of ODEs based on the law of mass action to a single
% equation describing the rate of substrate consumption and/or product
% formation (same magnitude opposite signs under quasi-steady state).
% This form describes the rate of change of substrate as a function of
% substrate concentration 's' ('t' is included for ODE solvers only).
% See EnzymeSubstrateProduct() description for explanation of 'params'.
% For clarity, rename parameters
kf = params(1);
kr = params(2);
kcat = params(3);
Eo = params(4);
 
% Define the differential equation describing d[S]/dt
% This form was taken from Schnell & Mendoza, 1997
Vmax = kcat*Eo;
KM = (kr + kcat)/kf; % Michaelis–Menten rate constant
dSdt = -Vmax*s/(KM + s);
end
Below, the Michaelis-Menten function above is solved numerically for [S] out of convinience. Then, [S] and d[S]/dt are compared visually to those from the exact solution obtained earlier by numerically solving the full mass action ODE system:

The plots above compare [S] from the Michaelis-Menten approximation against the exact solution to the full system of ODEs. These plots were all edited/formatted after-the-fact from the same MATLAB figure to show different time windows. As seen in the first plot, the Michaelis-Menten approximation holds relatively well for tc < t < ts. The rate of change of [S] in this range is nearly perfectly predicted by Michaelis-Menten (i.e. the lines are more or less parallel), and the actual value of [S] has only a slight offset/error. The source of this error in [S] can be seen in the second plot. Michaelis-Menten makes the assumption that the initial concentration of subtrate (S0) remains effectively unchanged after the initial transient/saturation of enzyme is complete [3]. As seen in the middle plot, there is a small but noticeable drop in substrate concentration for these parameters and initial conditions and the slight offset/error of the Michaelis-Menten-predicted [S] is simply due to not taking this into account. Based on everything discussed earlier from the paper by Schnell & Mendoza, I would expect that for higher values of S0 and/or KM (relative to E0), this slight difference would become even more negligible. The final plot above shows what happens around when the slow timescale comes to an end. While the slopes of [S] do begin to deviate a little more noticeably, they do not deviate very strongly, and the actual values of [S] converge as the reaction moves closer to completion. Thus, it seems that even around and after ts, Michaelis-Menten can still be a decent approximation depending on the level of accuracy needed (i.e. for any t sufficiently greater than tc).
Below are the corresponding comparison plots for d[S]/dt, zoomed-in around different time windows:
Consistent with my observations from the [S] plots, the Michaelis-Menten approximation is excellent at predicting the rate of reaction for a period within tc < t < ts for these parameters and initial conditions (top plot). This accuracy drops a bit as t approaches ts, and then it gets better again as the approximation and exact solution converge toward the final equilibrium (i.e. reaction completion). The only place where the Michaelis-Menten approximation is quite innacurate for these parameters and initial conditions is during the initial transient around tc (bottom plot). This makes sense, because the quasi-steady state assumption of d[ES]/dt ≈ 0 is inherently impossible when the free ezyme is still in the process of saturating with substrate. That said, this initial period is over extremely quickly relative to the time taken for reaction completion (compare timescales of top and bottom plots). Again, for any time sufficiently greater than tc, the Michaelis-Menten approximation still predicts the reaction rate (i.e. absolute value of d[S]/dt) at least somewhat accurately here, with the highest accuracy occuring during a period within tc < t < ts (sufficiently far from either boundary). Even when t approaches and passes ts, the accuracy could certainly be much worse.
As stated earlier, I would expect that as S0 gets even larger relative to fixed values of E0 and KM, the Michaelis-Menten approximation would become even more accurate and the minor discrepencies seen above would diminish. This can be shown by re-running everything with a larger value of S0 and leaving all else equal. All of the the code shown earlier was run again using S0=500, and the resulting plots comparing the exact solution to Michaelis-Menten can be seen below:
As seen above, the already small error in [S] obtained by solving Michaelis-Menten earlier is now much smaller relative to the magnitude of [S]. For S0=100, d[S]/dt from Michaelis-Menten was quite accurate for times sufficiently beyond tc and sufficiently before ts. In fact, while less accurate, it was not even that bad of an approximation for the reaction rate for times around or after ts. That said, for this larger value of S0, the accuracy of the reaction rate predicted by Michaelis-Menten is notably improved for all times sufficiently beyond tc, even though some minor discrepency still remains for times around ts (right plot). None of this is too surprising, as the assumptions upon which Michaelis-Menten is based are met more strongly as S0 increases (and all else remains equal). The only area where the reaction rate predicted by Michaelis-Menten is highly innacurate, even above for S0=500, is the initial transient around and before tc. This is not shown above for S0=500, but it was quite simlar in character to the transient regime shown earlier for d[S]/dt and S0=300. This makes sense because, by its very nature, the initial enzyme saturation process cannot be described as a quasi-steady state, because the enzyme, substrate, and complex concentrations are all changing at a very rapid rate (even though the difference between S0 and [S] at t = tc is still very small).
I was curious about what the simulation and comparisons would look like when the assumptions of the approximation are much farther from accurate, so I re-ran everything using S0 = 1. The plots summarizing the corresponding results can be seen below:
As seen in the first plot at the top, under the exact solution, the complex concentration [ES] is never really constant and the initial saturation has a major impact on [S] during 0 < t < tc. As such, the values of [S] predicted by solving Michaelis-Menten are far less accurate than those seen earlier (bottom right). The rate of reaction, as indicated by d[S]/dt, is now also much less accurate across all times. As expected, we still see convergence of the solutions from either method as the reaction nears completion. That said, Michaelis-Menten is much less accurate along the way for this lower value of S0. Under these circumstances, it seems that Michaelis-Menten is actually most accurate for t > ts, unlike earlier where it acheived the highest accuracy during a period within tc < t < ts for high relative values of S0.
